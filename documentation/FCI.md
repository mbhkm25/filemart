FileMart — Component Inventory (FCI)

Markdown Version — Arabic
Version 1.0

1. مقدمة

يهدف هذا الملف إلى تحديد جميع مكوّنات الواجهة (UI Components) داخل منصة FileMart مع وصف أماكن وجودها، وظائفها، حدود استخدامها، وسلوكها التفاعلي.

يساعد هذا المستند على:

منع إعادة تصميم المكوّنات

منع التكرار

ضمان الاتساق عبر كامل المنصة

تسهيل استخدام الذكاء الصناعي أثناء التطوير

بناء واجهة قابلة للتوسع على المدى الطويل

2. التصنيف العام للمكوّنات

تنقسم مكوّنات FileMart إلى:

المكوّنات العامة (Shared UI Components)

مكوّنات الواجهة العامة (Public Components)

مكوّنات لوحة التاجر (Dashboard Components)

مكوّنات لوحة المدير (Admin Components)

مكوّنات مساعدة (Helper Components)

3. المكوّنات العامة (Shared UI Components)
3.1 Button

الاستخدامات:

الإجراءات الأساسية

الحفظ

الإضافة

الحذف

الأنواع:

Primary

Secondary

Outline

Danger

القيود:

لا يُسمح بتغيير اللون الأساسي

لا يُسمح بتغيير الـradius

لا استخدام للظلال الثقيلة

3.2 Input

الوصف:
حقل الكتابة الأساسي بأنواعه (Text, Email, Number)

القيود:

لا أيقونات داخلية مدمجة

يجب وضع رسائل الخطأ أسفل الحقل

3.3 Card

الاستخدام: عرض المحتوى ضمن حاوية معزولة.

الخصائص:

padding: 16px

radius: 12–16px

ظل خفيف

3.4 Modal

يُستخدم للعمليات التداخلية (Interruptive Actions):

الحذف

التأكيد

التحرير الجزئي

3.5 NavbarMobile / NavbarDesktop

الاستخدام:
العنصر الأساس للتنقل عبر الواجهة.

القيود:

روابط واضحة

منع الظلال الثقيلة

3.6 Skeleton

الاستخدام:
العرض أثناء التحميل.

القيود:

لون ثابت

شكل ثابت

حركة Pulse فقط

4. مكوّنات الواجهة العامة (Public Components)
4.1 BusinessHeader

يعرض:

الاسم

الوصف

الغلاف

اللوجو

القيود:

لا منطق داخلي

مكوّن للعرض فقط

4.2 ProductCard

الاستخدام: عرض بيانات المنتج داخل الملف التجاري.

الحقول:

صورة

اسم

سعر

التفاعل:

الضغط: فتح صفحة المنتج

hover: تكبير بسيط

4.3 GalleryGrid

الوصف: شبكة صور الواجهة العامة.

القيود:

لا تعديل

مجرد عرض

4.4 PluginWidget

وحدة عرض الإضافات العامة.

القيود:

تصميم بسيط

ملتزم بدليل التصميم الأساسي

اسم

سعر

زر تعديل

5.4 PluginBox
